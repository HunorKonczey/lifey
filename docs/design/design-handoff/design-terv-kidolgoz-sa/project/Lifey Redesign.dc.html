Handoff notes
• All tokens → real ColorScheme.dark/.light + ThemeData in app_theme.dart; semantic tokens in app_tokens.dart.

• Build adaptive_bottom_nav.dart + adaptive_app_bar.dart from one shared collapse signal (UserScrollNotification). Expanded ↔ pill shown on screens 01 vs 02.

• Icons = real Icons.* names; outline = unselected, filled = selected.

• Charts reuse the custom TimeSeriesChart (screens 04–05). No new deps.

• English copy shown; ARB keys already exist — don't hardcode strings.
