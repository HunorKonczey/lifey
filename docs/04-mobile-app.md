# Mobile Application Requirements

## Screens

### Dashboard

Display:

* Today's calories
* Today's macros
* Latest weight
* Recent workouts

### Nutrition

Features:

* Food list
* Recipe list
* Meal logging

### Recipes

Features:

* Create recipe
* Edit recipe
* Delete recipe

### Workout Templates

Features:

* Create template
* Edit template

### Workout Tracking

Features:

* Start workout
* Add exercise
* Add set
* Finish workout

### Weight Tracking

Features:

* Add weight
* View history

## Navigation

Bottom navigation:

* Dashboard
* Nutrition
* Workouts
* Weight

## Offline Support

User should be able to:

* View data
* Add data
* Edit data

without internet connection.

## UI Priorities

* Fast interaction
* Minimal clicks
* Mobile-first experience

Design can remain simple during MVP.
