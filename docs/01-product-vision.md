# Fitness & Nutrition Tracker

## Goal

Build a personal fitness and nutrition tracking application.

The application should allow a user to:

* Track daily calories
* Create and manage recipes
* Track workouts
* Track body weight
* View basic statistics and progress over time

This is initially a personal project for a single user but should be designed in a way that supports multiple users in the future.

## Target Platforms

Phase 1:

* iOS

Phase 2:

* Android

The application should use a shared codebase whenever possible.

## Non Goals (Phase 1)

The following features are intentionally excluded from the first version:

* Social features
* AI-generated recipes
* AI calorie estimation from photos
* Barcode scanning
* HealthKit integration
* Garmin integration
* Strava integration
* Calculate carbohydrates
* Calculate fat

## MVP Features

### Nutrition

* Create foods manually
* Create recipes manually
* Log meals
* Calculate calories
* Calculate protein

### Workouts

* Create workout templates
* Log workout sessions
* Track exercises
* Track sets
* Track reps
* Track weight

### Weight Tracking

* Daily body weight entry
* Weight history

### Dashboard

* Daily calorie summary
* Daily macro summary
* Recent workouts
* Current body weight

## Design Philosophy

The application should prioritize:

* Simplicity
* Fast data entry
* Offline-first thinking
* Clean architecture
* Easy future extensibility
